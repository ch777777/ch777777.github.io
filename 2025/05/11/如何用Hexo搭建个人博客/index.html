常用命令

  • hexo server - 启动本地服务器预览博客
  • hexo new page "页面名" - 创建新页面
  • hexo clean - 清除缓存文件和已生成的静态文件
  • hexo generate - 生成静态文件
  • hexo deploy - 部署网站
